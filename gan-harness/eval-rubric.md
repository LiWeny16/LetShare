# Eval Rubric — PRO Upgrade Dialog

## Design Quality (weight: 0.35)
- Visual hierarchy: PRO card clearly stands out from Free
- Color harmony: palette consistent with LetShare brand
- Typography: clean sizing, proper weight contrast
- Spacing: comfortable breathing room, not cramped
- Premium feel: PRO tier looks desirable but not tacky

## Originality (weight: 0.30)
- Distinctive: doesn't look like a generic SaaS pricing page
- Creative: uses layout, gradients, or iconography in fresh ways
- Memorable: user would remember this dialog

## Craft (weight: 0.25)
- Pixel-perfect alignment
- Border-radius consistency
- Animation/transition consideration
- Icon usage appropriate and meaningful
- Responsive to dialog width

## Functionality (weight: 0.10)
- All existing functionality preserved
- TypeScript compiles without errors
- Copy-to-clipboard works
- Activation flow works
- Dialog opens/closes properly
